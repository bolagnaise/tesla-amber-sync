Custom TOU schedules allow you to configure fixed-rate electricity plans from providers like Origin Energy, AGL, Energy Australia, etc. that don't change dynamically like Amber.

Unlike Tesla's built-in rate plan creator which limits you to 4 rate periods (Super Off-Peak, Off-Peak, Mid-Peak, Peak), custom schedules support:

  • Unlimited time periods - perfect for complex TOU plans
  • 30-minute granularity - precise rate scheduling
  • Multiple seasons - summer/winter rate variations
  • Demand charges - for plans with capacity-based fees
  • Day-of-week control - weekday vs weekend rates

How It Works
  1. Create a Schedule - Define your electricity plan name, utility, and basic charges
  2. Add Seasons - Define time periods throughout the year (e.g., Summer: Dec-Feb, Winter: Jun-Aug)
  3. Add Time Periods - For each season, add rate periods with buy/sell rates and time ranges
  4. Preview - Check the generated Tesla tariff format before syncing
  5. Sync to Tesla - Send the schedule to your Powerwall (one-time setup)

Note: Unlike Amber integration which updates every 5 minutes, custom schedules are designed to be set once and only updated when your electricity provider changes rates.
